Install the Chrome Extension

The Wysely Chrome extension syncs your FanDuel and DraftKings bet history to the Wysely app automatically. Follow the steps below to install it.

Download Wysely Extension

Version 1.1.2 · Chrome · ~200 KB

Installation Steps

1

Download & Unzip

Click the download button above. Once downloaded, unzip the file — you should see a folder containing the extension files. Make note of where this folder is located on your machine.

2

Open Chrome Extensions

In Google Chrome, navigate to chrome://extensions by typing it in the address bar and pressing Enter.

3

Enable Developer Mode

In the top-right corner of the Extensions page, toggle Developer mode to on.

4

Load the Extension

Click "Load unpacked" in the top-left corner. Select the unzipped extension folder (the one containing manifest.json).

5

Pin the Extension

Click the puzzle piece icon in Chrome's toolbar, then click the pin icon next to Wysely so it's always visible.

6

Sign In & Sync

Click the Wysely icon in your toolbar, sign in with the same account you use in the app, then visit your sportsbook's My Bets page. Your bets will sync automatically.

Frequently Asked Questions

Is this safe?

Yes. The extension only reads your existing bet history from your sportsbook account — it cannot place bets, access your payment information, or modify anything. The source code is fully contained in the download.

Why isn't this on the Chrome Web Store?

The Chrome Web Store does not permit extensions that interact with sports betting websites, even read-only analytics tools like Wysely. Self-hosting allows us to distribute the extension directly to our users.

Chrome shows a "Disable developer mode extensions" popup. Is that normal?

Yes. Chrome shows this notification for any extension installed outside the Web Store. You can safely dismiss it — it will appear occasionally when you restart Chrome.

How do I update the extension?

Download the latest version from this page, unzip it, and replace the old extension folder. Then go to chrome://extensions and click the refresh icon on the Wysely card.

Which sportsbooks are supported?

Wysely currently supports FanDuel and DraftKings, with more sportsbooks planned.
